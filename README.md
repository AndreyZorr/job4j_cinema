job4j_cinema

В этом задании нужно разработать сайт по покупке билетов в кинотеатрю

Задача написать:
1.Регистрацию/Вход;
2.Вывод киносеансов и фильмов;
3.Покупку билетов.

Для реализации нужно использовать: Spring Boot, Thymeleaf, Bootstrap, Liquibase, Sql2o, PostgreSQL.

Требования к окружению: Java19, Sql2o 1.6.0, Liquibase 4.28.0.
